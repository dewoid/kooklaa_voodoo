# kooklaa_voodoo

### 1. Clone
```
$ git clone https://github.com/dewoid/kooklaa_voodoo.git
```

### 2. Create venv
```
$ python -m venv venv
```

### 3. Activate
```
# Linux / Mac OS
$ source venv/bin/activate

# Windows
$ venv\Scripts\activate
```

### 4. Install dependencies
```
$ pip install -r requirements.txt
```

### 5. Start app
```
$ python main.py
```
